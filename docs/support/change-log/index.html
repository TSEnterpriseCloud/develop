Version: 1.0.1

Release History

List of released implementations

2025​

2025.001.000​

  • Default Company field in Data Transfer Objects (DTO);
  • New service for importing and exporting customers and/or suppliers, go to casi d'uso

Master data services, go to use casi d'uso;

  • Region (RegionCO);
  • Province (ProvinceCO);
  • Bill of Materials (BomPD);
  • Transport Reasons (TransportReasonMG);

2024​

2024.003.000 (dec 2024)​

  • Service for reprocessing documents in error status following Bulk Document Insertion, go to casi d'uso;;
  • Service for Order Portfolio Fulfillment by item with variant filter, go to casi d'uso;
  • Import of an item price list from an external third-party file, go to casi d'uso;
  • Import of supplier orders and their fulfillment, go to casi d'uso;

Master data services, go to casi d'uso;

  • Collection of services for updating and reading additional warehouse data for Customer/Supplier: CustomerSupplierExtensionMG and CustomerSupplierMG;
  • Carriers and Stock Availability (CarrierMG, GiacenzaMG);
  • Contact Management (ContactLinksCO);
  • Contact Type Management (ContactTypeCO);
  • Contact Subjects Management (ContactSubjectCO);
  • Batch Management for items (BatchDataWH);
  • Fiscal Groups (Fiscal GroupWH);
  • Macro Areas (MacroAreaCO);
  • Areas (AreaCO);
  • Zones (ZoneCO);
  • Brands (BrandWH);
  • Statistical Group 1 (StatisticalGroup1WH);
  • Statistical Group 2 (StatisticalGroup2WH);
  • Statistical Group 3 (StatisticalGroup3WH);
  • Statistical Group 4 (StatisticalGroup4WH);
  • International measurement units (UMCodeCO);
  • Company measurement units (UnitOfMeasureCO);
  • Administrative offices (OfficePACO);
  • Families (FamilyWH);
  • Sub Families (SubFamilyWH);
  • Groups (ItemGroupWH);
  • Sub Groups (ItemSubGroupWH).

2024.002.000 (jul 2024)​

  • New service for generating document print on pdf, go to [casi d'uso](../support/examples-apis# https://ts-enterprise-dev.ts-paas.com/link/23#bkmrk-page-title);
  • New Service for inserting and exploding kits, go to casi d'uso;
  • New Service for Document Transformation Management (1 to 1 Documents);
  • New Service for order fulfillment status;

go to casi d'uso;

  • New Service for Order Portfolio Fulfillment;
  • Document List Lookup (Borderaux);
  • DDT Fulfillment and Invoice Generation (1 to n Documents);

go to casi d'uso;

  • New Service for Bulk Document Insertion, go to casi d'uso;
  • Common master data services and company master data (NationCO, LanguageCO, AgencyCO and BankCO, Office CO), go to casi d'uso;

2024.001.000 (mar 2024)​

  • VAT Journal Entries Management, go to casi d'uso;
  • Non-VAT Journal Entries Management, go to use cases;
  • Transfer Journal Entries Management, go to use cases;
  • Receipts/Payments Journal Entries Management, go to use cases;
  • VAT codes master data service, go to casi d'uso;
  • Priority price lists management, go to casi d'uso;
  • Implemented the possibility to insert zero quantity on the line with the PUT document;
  • Packaging management on item-type document lines;

2023​

Release versions and related features​

2023.003.000 (dec 2023)​

  • Implemented project line types management in documents: Node (15), Activity (16), Project Expense (17);
  • Changed data type of SearchCount(SimpleSearchRequestInfo expression) method from int to long;
  • Implemented in document writing the generation of the linked document with another warehouse, if configured in document customization;
  • New service for intra-item data loading;
  • Valuation of "TSDIGITAL Routing" parameters;
  • Implemented the possibility to have more than 50 records per page in search-type webapi responses and the total number of records to allow pagination;
  • Implemented reading/writing of mandatory fields for projects and nodes;
  • Implemented the possibility to manage the season and line field both at line and header level;
  • New service for managing the Agent - Items entity;
  • Implemented the possibility to load the asset leaving the BO to assign the code;

2023.002.000 (jul 2023)​

  • Added multiple packaging on items;
  • New service for calling prices and discounts generated by parametric price lists;
  • Implementation to register documents with mechanisms that allow acquiring variant grid data;
  • Implementation of extended attributes on DTO Projects;
  • Implemented a new "disableLetterOfIntent" parameter in "callOptions" to exclude the letter of intent line from the document;
  • Implemented the possibility to insert grouped lines on the document to manage them in grid;
  • Implemented the call for Families-Subfamilies Groups-Subgroups entities;
  • Implemented intra management in documents, go to casi d'uso;

2023.001.000 (mar 2023)​

  • Multiple Agents and Preferred Agent management on Customer WebAPI;
  • Implemented the possibility to insert "Fixed Text" lines with the exposure in the document lines of the new "FixedTextMG" property;
  • Project management in document insertion;
  • VAT exemption code parameterization in document insertion, go to casi d'uso;;
  • Customer lookup exposure for search by VAT number and/or tax code, go to casi d'uso;

2022​

Release versions and related features

2022.003.000 (jan 2022)​

  • First Cloud release of a limited subset of web services to support the scenario of Customer Order insertion from an external B2B e-commerce;
