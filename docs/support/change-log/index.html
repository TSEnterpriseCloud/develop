Release History

List of released implementations

2025​

2025.001.000​

  • Registry services, go to supported use case;
  • Region (RegionCO);
  • Province (ProvinceCO);

2024​

2024.003.000 (dec 2024)​

  • Service for reprocessing documents in error state following Mass Document Insertion, go to supported use case;
  • Order Portfolio Fulfillment Service by item with variant filter, go to supported use case;
  • Import of an item price list from an external third-party file, go to supported use case;
  • Import of supplier orders and their fulfillment, go to supported use case;

Registry services, go to supported use case;

  • Collection of services for updating and reading additional warehouse data for Customer/Supplier: CustomerSupplierExtensionMG and CustomerSupplierMG;
  • Carriers and Stock Availability (CarrierMG, StockMG);
  • Contact Management (ContactLinksCO);
  • Contact Type Management (ContactTypeCO);
  • Contact Subjects Management (ContactSubjectCO);
  • Batch Management by item (BatchDataWH);
  • Fiscal Groups (FiscalGroupWH);
  • Macro Areas (MacroAreaCO);
  • Areas (AreaCO);
  • Zones (ZoneCO);
  • Brands (BrandWH);
  • Statistical Group 1 (StatisticalGroup1WH);
  • Statistical Group 2 (StatisticalGroup2WH);
  • Statistical Group 3 (StatisticalGroup3WH);
  • Statistical Group 4 (StatisticalGroup4WH);
  • International measurement units (UMCodeCO);
  • Company measurement units (UnitOfMeasureCO);
  • Administrative offices (OfficePACO);
  • Families (FamilyWH);
  • Sub Families (SubFamilyWH);
  • Groups (ItemGroupWH);
  • Sub Groups (ItemSubGroupWH).

2024.002.000 (jul 2024)​

  • New document print generation service in pdf, go to supported use case;
  • New Service for kit insertion and explosion, go to supported use case;
  • New Service for Document Transformation (1 to 1 Documents);
  • New Service on Order Fulfillment Status;

Go to supported use case;

  • New Order Portfolio Fulfillment Service;
  • Document List Lookup (Borderaux);
  • DDT Fulfillment and Invoice Generation (1 to n Documents);

Go to supported use case;

  • New Mass Document Insertion Service, go to supported use case;
  • Common registries and company registry services (NationCO, LanguageCO, AgencyCO and BankCO, Office CO), go to supported use case;

2024.001.000 (mar 2024)​

  • Management of VAT Journal Entries, go to supported use case;
  • Management of Non-VAT Journal Entries, go to supported use case;
  • Management of Transfer Journal Entries, go to supported use case;
  • Management of Receipts/Payments Journal Entries, go to supported use case;
  • VAT code registry service, go to supported use case;
  • Management of priority price lists, go to supported use case;
  • Implemented the possibility to insert zero quantity on the line with the PUT document;
  • Management of packaging on item document lines;

2023​

Release versions and related features​

2023.003.000 (dec 2023)​

  • Implemented in documents the management of project line types: Node (15), Activity (16), Project Expense (17);
  • Changed data type of method SearchCount(SimpleSearchRequestInfo expression) from int to long;
  • Implemented in document writing the generation of the linked document with another warehouse, if configured in document customization;
  • New service for intra item data loading;
  • Valuation of "TSDIGITAL Routing" parameters;
  • Implemented the possibility to have more than 50 records per page in the response of search-type webapis and the total number of records to allow pagination;
  • Implemented reading/writing of mandatory fields for projects and nodes;
  • Implemented the possibility to manage the season and line field both at line and header level;
  • New service for managing the Agent - Items entity;
  • Implemented the possibility to load the asset leaving the BO the task of assigning the code;

2023.002.000 (jul 2023)​

  • Added multiple packaging on items;
  • New service for calling prices and discounts generated by parametric price lists;
  • Implementation to be able to register documents with mechanisms that allow acquiring "grid" variant data;
  • Implementation of extended attributes on Project DTOs;
  • Implemented a new parameter "disableLetterOfIntent" in "callOptions" to exclude the letter of intent line from the document;
  • Implemented the possibility to insert grouped lines in the document to manage them in a grid;
  • Implemented the call for Family-Subfamily Group-Subgroup entities;
  • Implemented intra management in documents, go to supported use case;

2023.001.000 (mar 2023)​

  • Management of Multiple Agents and Preferred Agent on Customer Webapi;
  • Implemented the possibility to insert "Fixed Text" type lines with the exposure of the new "FixedTextMG" property in the document lines;
  • Project management in document entry;
  • Parameterization of VAT exemption code in document entry, go to supported use case;
  • Exposure of customer Lookup for search by VAT number and/or tax code, go to supported use case;

2022​

Release versions and related features

2022.003.000 (jan 2022)​

  • First Cloud release of a limited subset of web services supporting the Customer Order entry scenario from an external B2B e-commerce;
